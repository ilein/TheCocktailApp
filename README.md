# TheCocktailApp

## Описание
Поиск разных коктейлей по названию и просмотр понравившихся  
API: https://thecocktaildb.com/api.php

## Используется
MVI  
  
- RxJava  
- Coroutines  
- Ktor  
- Room  
- Koin  
- Coil  
- Android Navigation Component
